# Servicios Backend

Servicios REST desarrollados con Flask para la gestión de plantillas y generación de documentos.

## Requisitos

- Python 3.9+
- Docker
- PostgreSQL

## Estructura del proyecto

```
services/
├── app/
│   ├── __init__.py
│   ├── config.py
│   ├── models/
│   ├── routes/
│   ├── services/
│   └── utils/
├── tests/
├── Dockerfile
├── requirements.txt
└── run.py
```
